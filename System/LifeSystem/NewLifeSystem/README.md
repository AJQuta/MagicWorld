# LifeSystem Structure

## Base Structure

- LifeForm
    - Animal
        * No Inventory
        * Cannot Attack
    - Fighter
        * Can Attack
        - Player
            * Has an Inventory
            * Has Knowledge
        - Enemy
            * Has a Weapon
            - Boss
            - MiniBoss [very low priority]
    - NPCs
        - QuestNPCs
        - GenericNPCs
        - ExchangeNPCs
